obtive alguns desafios com este sistema:

		1 - comunicação com banco MySql, eu estava
		inserindo as autenticações invertidas, então
		o sistema procurava primeiro a senha e não o usuário,
		gerando um erro de permissão:
		
			Errado: connection = DriverManager.getConnection(banco, password, user);
				correto: connection = DriverManager.getConnection(banco, user, password);
		
		2 - Erro de timezone do banco, 
		resolvido com a inserção deste atributo:
				
				problema: private static String banco = "jdbc:mysql://localhost:3306/admin_combo";		
					solução: private static String banco = "jdbc:mysql://localhost:3306/admin_combo?useTimezone=true&serverTimezone=UTC";
